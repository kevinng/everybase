How do extra credits to contact users work?

{% comment %} End: Heading {% endcomment %} {% comment %} Start: Body {% endcomment %}
Extra credits in your account are only deducted when you contact users above your daily limit. E.g., if your daily limit is 100 users/day, and you've only used 35 credits - extra credits in your account will not be deducted. Extra credits have an expiry of 90 days from the date of purchase.

How do I contact a user? And how are contact user credits deducted?

{% comment %} End: Heading {% endcomment %} {% comment %} Start: Body {% endcomment %}
Each user on Everybase has a verified WhatsApp account. By clicking on a user's WhatsApp button, you may be redirected to the user's WhatsApp. A contact user credit is deducted when you're redirected to the user's WhatsApp the first time for the day. Contacting the same user multiple times in the same day will not cause credits to be deducted multiple times.

What is an alert? And what is an alert notification? And how are alert notification credits deducted?

{% comment %} End: Heading {% endcomment %} {% comment %} Start: Body {% endcomment %}
You create an alert to receive an alert notification via WhatsApp when a user updates requirements, and that requirements matches your needs. There is no limit on the number of alerts you can create. All plans have a limit on the number of alert notifications you may receive. An alert notification credit will be deducted when you receive an alert notification via WhatsApp.

What happens if I hit the alert notification limit for the month?

{% comment %} End: Heading {% endcomment %} {% comment %} Start: Body {% endcomment %}
All plans have a limit for the number of alert notifications you may receive in a month. When you hit the limit, you'll not be able to receive anymore alert notifications that month. You may either upgrade your plan, purchase extra credits, or wait for your credits to refresh the next month. You'll miss all user requirements updates that matches your alert settings that happens when you're out of credits.
